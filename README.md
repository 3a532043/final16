從github上clone回來
```
git clone https://github.com/WISD-2018/final16
```
切換到專案目錄
```
cd final16
```
還原composer 套件
```
composer install
```
還原npm套件
```
npm install
```
複製.env.example生成.env
```
cp .env.example .env
```
產生金鑰
```
artisan key:generate
```
測試
```
artisan serve
```
能成功看到主頁代表成功
